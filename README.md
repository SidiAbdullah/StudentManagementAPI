# Student Management API

A secure RESTful API built with ASP.NET Core demonstrating backend best practices including authentication, authorization, rate limiting, logging, and auditing.

Technologies Used: ASP.NET Core Web API, C#, JWT Authentication, Authorization Policies, Rate Limiting, Logging & Auditing, Swagger, In-Memory Database.

Features:
- Secure JWT token-based authentication
- Role-based and policy-based authorization
- Rate limiting to protect the API
- Logging and auditing system
- RESTful API endpoints
- Swagger UI for testing

Project Structure:
- Controllers handle HTTP requests
- Services contain business logic
- Middleware handles logging, auditing, and rate limiting

How to Run:
1. Clone the repository: `git clone https://github.com/SidiAbdullah/StudentManagementAPI.git`
2. Open the project in Visual Studio
3. Run the application
4. Open Swagger UI in your browser to test the API

Purpose:
This project demonstrates building a secure backend API with ASP.NET Core, showcasing key backend concepts used in professional projects.
